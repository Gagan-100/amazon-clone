# 🛒 Amazon Clone

A modern Amazon-inspired e-commerce web application built using modular JavaScript.  
This project replicates core shopping features like product browsing, cart management, checkout flow, and order tracking.

---

## 🚀 Live Demo

👉 https://amazon-clone-gagan.netlify.app/

---

## ✨ Features

- 🛍️ Product listing with dynamic rendering  
- 🔍 Search functionality  
- 🛒 Add to cart & update quantity  
- 💳 Checkout system  
- 📦 Order history  
- 🚚 Order tracking system  
- 📱 Fully responsive design  

---

## 🧠 Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6 Modules)  
- LocalStorage  

---
